## Available Tools (Context)
The following tools are available for use:
- `list_directory`: To list files and subdirectories.
- `read_file`: To read file content.
- `search_file_content`: To search for patterns within files.
- `glob`: To find files matching glob patterns.
- `replace`: To replace text within a file.
- `write_file`: To write content to a file.
- `run_shell_command`: To execute shell commands.
- `save_memory`: To save specific facts to memory.
- `google_web_search`: To perform web searches.
- `write_todos`: To manage a list of subtasks.
- `codebase_investigator`: For complex codebase analysis.
- `web_fetch`: To process content from URLs.
